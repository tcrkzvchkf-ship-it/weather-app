# Weather App

A command line weather app built with Python that fetches 
real-time weather data from the wttr.in API.

## Features
- Search weather for any city in the world
- Shows temperature, feels like, condition, humidity and wind speed
- Handles invalid cities gracefully
- Runs continuously until user quits
- Favrotie City feature

## How to Run

Install dependencies:
```bash
pip3 install requests
```

Run the app:
```bash
python3 weather.py
```

## Example Output

--- Weather for Orlando ---
Temperature:  69°F
Feels like:   69°F
Condition:    Clear
Humidity:     73%
Wind speed:   5 mph

## Tech Stack
- Python 3.11+
- requests library
- wttr.in API (free, no key required)
